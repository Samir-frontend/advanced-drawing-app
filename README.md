## 🖌️ About

An **advanced drawing application** built with the HTML5 **Canvas API** — lets users freely sketch, draw shapes, pick colors, and adjust brush sizes, just like a mini digital whiteboard.

## ⚙️ How It Works

```javascript
canvas.addEventListener('mousedown', startDrawing)
canvas.addEventListener('mousemove', draw)
canvas.addEventListener('mouseup', stopDrawing)

// Tracks mouse/touch coordinates
// Draws smooth lines using Canvas 2D context
// Supports color, brush size & eraser modes
```
